
# GlobalStore Sales Data Analysis

This project contains an Exploratory Data Analysis (EDA) on GlobalStore sales data to understand trends, patterns, and insights.

##  Key Objectives

- Explore sales trends over time  
- Identify top-selling products and categories  
- Analyze customer purchasing patterns  
- Visualize seasonal or regional sales variations  
- Prepare insights to support business decisions  

##  Tools & Technologies

- **Python**: `pandas`, `numpy`, `matplotlib`, `seaborn`  
- **Excel**: for initial data cleaning and simple visualizations  
- **Jupyter Notebook**: for interactive analysis and charts  


## Key Insights (Example)

- Top 5 products contribute **X% of total sales**  
- Seasonal peaks in sales during **specific months**  
- Certain product categories perform better in **specific regions**  
- Sales trends show **consistent growth or decline patterns** over time
